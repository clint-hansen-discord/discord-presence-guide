Rich Presence — User Privacy Controls

A walkthrough of the controls every Discord user has to manage what game activity is shared, with whom, and on what terms.

Activity sharing (Rich Presence) is on by default for all Discord users — but every player has full, self-service controls to manage visibility at the global, per-server, and per-game level. No linked account is required to access or change any of these settings.

User Expectations

Discord users understand and actively use presence. Sharing what you’re playing is a normal, expected part of how people present themselves on Discord — not a hidden or obscure setting.

  • Presence is a visible, everyday feature: it appears on your profile, in member lists, and on friends’ screens, so users are continuously aware their activity is being shown.
  • Controls are granular and easy to reach — visibility can be turned off globally, or limited per-server and per-game, all from within Settings.
  • Every privacy control lives directly in Settings and can be changed at any time, with no setup required.
  • Sensitive data is never exposed: presence shows the game and session basics, never in-game identity, rank, or account details.
1
Master Activity Toggle — Turn Off All Presence

The top-level switch. When disabled, Discord stops broadcasting any game or app activity to anyone — across all servers and friends.

Settings → Activity Privacy → Activity Sharing → “Share my activity”
Settings menu
Activity Privacy — master toggle (top right) and Default Sharing options below
2
Default Sharing Scope — Choose Who Sees Your Activity

Controls who sees presence by default when joining any new server. Three options:

Friends & All Servers (default)  ·  Friends & Small Servers Only (≤200 members)  ·  Friends Only

Settings → Activity Privacy → Default Sharing

Radio buttons visible in the Activity Privacy screenshot above.

3
Per-Server Overrides — Customize Each Server Individually

Players can override the default for specific servers — e.g. sharing with a small friends’ server while keeping activity hidden on a large public server. The My Servers list shows every server with an individual toggle.

Settings → Activity Privacy → My Servers
4
Per-Game Visibility — Hide Specific Titles

Even with global sharing on, players can disable detection for individual games via the eye icon next to each registered game. Toggling it off removes that title from presence without affecting other games.

Settings → Registered Games → eye icon per game
Detection on — League of Legends visible in presence
Eye toggled off — game removed from presence
5
Who Can Join My Games — Control Direct Join Access

Separate from visibility: players control whether friends or voice channel participants can join their session directly from the presence card. These toggles are independent — a player can be visible but not joinable.

Settings → Activity Privacy → Who Can Join My Games
Join controls — independent from what others can see
6
What Presence Looks Like to Others

When active, presence shows game name, mode, elapsed time, and party size on the player’s profile card and in server member lists. No linked account data, in-game identity, or rank is ever exposed.

“Now playing” indicator in Registered Games
Profile card — game, mode, and party size visible to others

What IS shared (when enabled)

  • Game name
  • Game mode / activity type
  • Elapsed session time
  • Party size (e.g. 1 of 5)

What is NEVER shared

  • In-game username or rank
  • Linked account details
  • Voice or chat content
  • Personal account data

Key Takeaways

  • Presence is opt-out, not opt-in — but disabling it takes a single toggle.
  • Controls are granular: global, per-server, and per-game visibility are all independently configurable.
  • The eye icon in Registered Games lets a player hide one specific game without touching anything else.
  • All controls live in Settings and are available to every Discord user, with no setup required.
  • Join controls are separate from visibility — a player can be visible but not joinable.
These privacy controls are part of Discord’s core settings and are available to every user from within the app.
